## 📝 Descripción del Proyecto
**Mind Health** es una plataforma diseñada para la gestión integral del bienestar mental y personal. Este proyecto se desarrolla como parte del curso de Desarrollo de Software, enfocándose en la creación de una arquitectura escalable, diseño de experiencia de usuario (UI/UX) y un modelo de datos robusto para soportar el seguimiento de metas, planes de suscripción y sesiones de bienestar.

## 🚀 Estado del Sprint 1

- [x] Configuración del repositorio y Git Flow (Rama `develop`).

- **Gestión de Dependencias:** Maven
- **Base de Datos:** PostgreSQL
- **Modelado:** IntelliJ IDEA / Bizagi
